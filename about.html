Lets Calculate MBG!

Website Informasi Makan Bergizi Gratis — program prioritas nasional.

Apa itu Lets Calculate MBG?

Lets Calculate MBG! adalah sebuah website yang saya buat menggunakan sistem kejar semalam, di bantu oleh open code demi keperluan tugas ahir pendidikan pancasila.

Tujuan Website

Website ini dibuat untuk memberikan perspektif tentang seberapa besar anggaran harian MBG (Rp 855.000.000.000) jika dibandingkan dengan harga barang dan program lain. Bukan untuk meremehkan, melainkan untuk membantu masyarakat memahami skala pendanaan program ini secara lebih konkret.

Three Tools

Live Burn Rate

Menampilkan akumulasi anggaran yang terpakai secara real-time sejak halaman dibuka, dengan laju Rp 9.895.833/detik.

Calculator

Hitung berapa lama anggaran harian MBG mampu membiayai suatu barang berdasarkan harganya.

Spend MBG

Simulasikan belanja anggaran harian untuk berbagai program seperti beasiswa, infrastruktur sekolah, dan layanan publik.

Data & Sumber

Anggaran Rp 855 miliar/hari adalah data realisasi resmi Badan Gizi Nasional per Januari 2026. Realisasi harian dapat berubah tergantung jumlah Satuan Pelayanan Pemenuhan Gizi (SPPG) dan penerima manfaat yang aktif.

Data Mode "SPEND" adalah kalkulasi sederhana tanpa banyak pemikiran, akan saya update jika ada waktu.

Teknis

Dibangun dengan HTML, Tailwind CSS, dan Vanilla JavaScript. Tanpa backend, tanpa dependency berat. Semua kalkulasi berjalan di sisi klien.

Data didasarkan pada laju pengeluaran program MBG sebesar Rp 855 Miliar/hari. Estimasi berdasarkan realisasi anggaran harian.
